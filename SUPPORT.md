# Support

Use [GitHub Issues](https://github.com/alexmorris10x/ios-boilerplate/issues) for reproducible boilerplate bugs and focused feature requests. Use [GitHub Discussions](https://github.com/alexmorris10x/ios-boilerplate/discussions) for broader implementation questions.

Include the Xcode version, iOS target, repository commit or release, steps to reproduce, and relevant redacted logs. This project is maintained on a best-effort basis with no guaranteed response time. Application-specific integration work and production operations are outside the support boundary.

Do not post credentials, production data, or vulnerability details publicly. Report security problems through [SECURITY.md](SECURITY.md).
